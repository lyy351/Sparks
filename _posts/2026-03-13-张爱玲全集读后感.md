## 张爱玲作品及生平

| 作品 | 创作/发表年份 | 张爱玲生平大事 | 时代背景与生活印记 |
| --- | --- | --- | --- |
|  | 1920年9月30日 | 出生于上海公共租界一幢没落贵族府邸，原名张煐。 | 显赫家世：祖父张佩纶是清末名臣，祖母李菊耦是李鸿章的长女。父亲是典型的遗少，母亲是新式女性。 |
|  | 1922年 | 随父母迁居天津。 | 开始接受私塾教育，家庭氛围新旧交织。 |
|  | 1924年 | 母亲与姑姑赴欧洲游学，童年的张爱玲由姨奶奶看管。 | 母亲的缺席，为她后来的"恋父/寻母"文学母题埋下伏笔。 |
|  | 1927年 | 开始写小说，第一部写家庭悲剧，第二部写女郎失恋自杀。 | 7岁起笔，已显文学天赋。 |
|  | 1928年 | 随父亲搬回上海，开始学绘画、英文和钢琴，读古典名著。 | 中西文化的双重熏陶，奠定她日后"新旧文学打成一片"的独特风格。 |
|  | 1930年 | 父母离婚，改名张爱玲（英文名Eileen的音译）。 | 家庭的破裂，让她过早体验人世的苍凉。 |
|  | 1931年 | 入读上海圣玛利亚女校。 | 教会学校的西式教育，拓展了她的国际视野。 |
| 《不幸的她》（短篇小说） | 1932年 | 就读于上海圣玛利亚女校，12岁在校刊《凤藻》发表处女作。 | 此时父母已离异，她随父亲生活，开始接受中西文化熏陶。 |
| 《迟暮》（散文） | 1933年 | 继续在圣玛利亚女校就读，陆续发表文章。 | 与父亲学写旧诗，广泛阅读《红楼梦》等古典文学。 |
| 《秋雨》（散文） | 1936年 | 就读圣玛利亚女校。 | 在校刊《凤藻》发表。 |
| 《牛》（短篇小说） | 1936年 | 就读圣玛利亚女校。 | 发表于《国光》创刊号。 |
| 《论卡通画之前途》（随笔） | 1937年 | 从圣玛利亚女校毕业。与父亲关系恶化，离家与母亲同住。 | 发表于《凤藻》。封建家庭与追求自由的冲突，成为她日后笔下""父权的崩塌""的雏形。 |
| 《牧羊者素描》（散文） | 1937年 | 从圣玛利亚女校毕业。 | 发表于《凤藻》。 |
| 《霸王别姬》（短篇小说） | 1937年 | 中学毕业，与继母发生争执，被父亲软禁数月，后离家出走投奔母亲。 | 发表于《国光》第九期。家庭矛盾激化，这段经历深刻影响了她日后的创作。 |
| 《天才梦》（散文） | 1939年 | 考取伦敦大学，因战事未能成行，改入香港大学文学系，专攻文学。 | 获《西风》杂志征文第十三名，后收入《张看》。母亲与姑姑赴欧洲游学，她在港大广泛阅读西方历史和文学。香港的华丽与悲哀成为她早期最重要的创作背景。 |
| 《沉香屑·第一炉香》（短篇小说） | 1943年 | 1941年太平洋战争爆发，港大停办，学业中断。战争经历让她对"时代的车轰轰地往前开"有了切身体验。战争中的香港，成为《沉香屑·第一炉香》《倾城之恋》等"香港传奇"系列的直接素材。于1942年返回上海，开始为英文报刊撰写影评和散文，在《泰晤士报》、《二十世纪》等英文刊物上发表影评、散文，英文写作的经历，练就了她精准、冷峻的笔调。 | 发表于周瘦鹃主编的《紫罗兰》杂志，一举成名。这是她登上文坛的成名作。叩响文坛的处女作。以香港为背景，讲述上海女孩葛薇龙在殖民地环境中的沉沦。 |
| 《沉香屑·第二炉香》（短篇小说） | 1943年 | 返回上海后，开始职业写作生涯。 | 发表于《紫罗兰》杂志。 |
| 《茉莉香片》（短篇小说） | 1943年 | 结识周瘦鹃、柯灵、苏青、胡兰成等文化界人士。 | 发表于《杂志》月刊第11卷4期。 |
| 《到底是上海人》（散文） | 1943年 | 在上海文坛开始崭露头角。 | 发表于《杂志》月刊。 |
| 《心经》（短篇小说） | 1943年 | 创作高峰期。 | 发表于《万象》月刊第2-3期。 |
| 《倾城之恋》（短篇小说） | 1943年 | 与胡兰成相识。 | 发表于《杂志》第11卷6-7期。 |
| 《琉璃瓦》（短篇小说） | 1943年 | 创作高峰期。 | 发表于《万象》第5期。 |
| 《金锁记》（短篇小说） | 1943年 | 一举奠定其在文坛的重要地位。 | 发表于《杂志》第12卷2期。傅雷以“迅雨”笔名发表评论，誉其为“我们文坛最美的收获之一”。 |
| 《封锁》（短篇小说） | 1943年 | 持续创作。 | 发表于《天地》月刊第2期。 |
| 《洋人看京戏及其他》（散文） | 1943年 | 创作高峰期。 | 发表于《古今》月刊。 |
| 《更衣记》（散文） | 1943年 | 创作高峰期。 | 发表于《古今》月刊。 |
| 《公寓生活记趣》（散文） | 1943年 | 创作高峰期。出版小说集《传奇》 （初版，奠定文坛地位的代表作。收录了上述大部分名篇。）出版散文集《流言》（展现她个人生活、审美与思想的散文集。） | 记趣发表于《天地》月刊。 |
| 《连环套》（中篇小说） | 1944年 | 与胡兰成结婚，炎樱为证婚人。这段短暂而备受非议的婚姻，是她人生中最大的争议点。 | 发表于《万象》7-10期，后收入《张看》。 |
| 《年青的时候》（短篇小说） | 1944年 | 创作巅峰期。 | 发表于《杂志》第12卷5期。 |
| 《花凋》（短篇小说） | 1944年 | 创作巅峰期。 | 发表于《杂志》第12卷6期。 |
| 《红玫瑰与白玫瑰》（短篇小说） | 1944年 | 创作巅峰期。 | 发表于《杂志》第13卷2-4期。 |
| 《鸿鸾禧》（短篇小说） | 1944年 | 创作巅峰期。 | 发表于《新东方》第9卷第6期。 |
| 《殷宝滟送花楼会》（短篇小说） | 1944年 | 创作巅峰期。 | 发表于《杂志》第14卷2期，后收入《惘然记》。 |
| 《等》（短篇小说） | 1944年 | 创作巅峰期。 | 发表于《杂志》第14卷3期。 |
| 《桂花蒸 阿小悲秋》（短篇小说） | 1944年 | 创作巅峰期。 | 发表于《苦竹》月刊第2期。 |
| 《留情》（短篇小说） | 1945年 | 抗战胜利前夕，时局动荡。 | 发表于《杂志》第14卷5期。 |
| 《创世纪》（短篇小说） | 1945年 | 8月15日，日军投降，抗战胜利。丈夫胡兰成作为汉奸潜逃。 | 发表于《杂志》第14卷6期，第15卷1、3期，后收入《张看》。 |
| 《苏青张爱玲对谈记》（散文） | 1945年 | 抗战胜利，胡兰成逃亡。 | 发表于《杂志》月刊。 |
| 《倾城之恋》改编为话剧 | 1945年 | 自编话剧《倾城之恋》在上海公演。 | 沦陷区特殊的文化环境下，她的作品被搬上舞台。 |
| 《多少恨》（短篇小说） | 1947年 | 与胡兰成离婚。与电影导演桑弧合作，开始创作电影剧本如《不了情》《太太万岁》，转向大众文化市场。 | 发表于《大家》月刊第2-3期，后收入《惘然记》。 |
| 《华丽缘》（散文） | 1947年 | 战后上海，文化环境发生变化。 | 发表于《大家》月刊创刊号，后收入《惘然记》。 |
| 《不了情》《太太万岁》（电影剧本） | 1947年 | 与桑弧合作，创作电影剧本。 | 文华影业公司出品，是战后上海电影的重要作品。 |
| 《十八春》（长篇小说） | 1948-1951年 | 以笔名“梁京”在上海《亦报》连载，（后改名《半生缘》）。1950年参加上海市第一届文学艺术界代表大会。 | 1949年上海解放，这是她第一部完整的长篇小说，，哀婉的都市爱情悲剧，试图向新时代靠拢。 |
| 《小艾》（中篇小说） | 1951年 | 在上海《亦报》发表。 | 上海政权更替，文化环境剧变。 |
| 《秧歌》（长篇小说） | 1954年 | 1952年移居香港，在美国新闻处工作，开启了此后数十年的海外生涯。出版带有政治倾向的长篇小说《秧歌》《赤地之恋》。 | 香港《今日世界》月刊连载，后出版英文本及中文本。这两部作品使她在很长一段时间内被大陆文学史"屏蔽"。 |
| 《赤地之恋》（长篇小说） | 1954年 | 在香港美国新闻处工作期间完成。 | 香港《今日世界》出版。 |
| 《传奇》改名《张爱玲短篇小说集》在香港出版 | 1954年 | 移居香港后，作品在香港由天风出版社重新出版。 | 成为少数在港台持续出版作品的大陆作家。 |
| 《五四遗事》（短篇小说） | 1957年 | 1955年离港赴美，希望打入英语世界的努力开始。1956年与剧作家赖雅在纽约结婚，获写作奖金，与第二任丈夫的婚姻，直到他1967年去世。 | 发表于台北《文学》杂志，后收入《惘然记》。 |
| 《情场如战场》《人财两得》《桃花运》《六月新娘》《南北一家亲》《小儿女》《南北喜相逢》《一曲难忘》等电影剧本，英文自传小说《雷峯塔》《易经》完成。 | 1957-1964年 | 为香港电懋影业公司编写大量电影剧本，以商业写作维持生计，同时进行严肃的英文创作。1961年唯一一次访问台湾，为写作《少帅》搜集资料。《少帅》 以张学良为原型，这部小说直到她去世后才整理出版。 | 这是她尝试打入大众文化市场，并维持生计的重要阶段。 |
| 《怨女》（长篇小说） | 1966年 | 1967年丈夫赖雅去世。1969年任职加州大学伯克利分校“中国研究中心”。 | 香港《星岛晚报》连载，后由台北皇冠出版社出版。根据《金锁记》重写，在《星岛晚报》连载，对早期代表作的重写与深化。是她在美国完成的重要长篇。 |
| 《半生缘》（长篇小说） | 1968年 | 1969年,任职加州大学伯克利分校"中国研究中心"。《半生缘》 在台湾出版。开始从创作转向学术研究。 | 根据《十八春》改写，删去特定政治内容，突出爱情悲剧，先在《皇冠》杂志刊出，后出版。 |
| 《红楼梦魇》（论文集） | 1977年 | 1975年完成自传小说《小团圆》，这部自传色彩极浓的小说，在她去世后才出版。定居洛杉矶，开始离群索居的晚年生活。 | 历时十年之久的红学研究结晶，由皇冠出版。周汝昌赞其“堪称雪芹知已”。 |
| 《色，戒》（短篇小说） | 1979年发表 | 晚年深居简出，整理旧作。 | 发表于台湾《中国时报·人间副刊》，后收入《惘然记》。 |
| 《海上花列传》译注（《海上花》《海上花落》） | 1981-1983年 | 晚年致力于翻译工作。 | 将吴语小说《海上花列传》译为国语并注释，由皇冠出版。 |
| 《惘然记》《余韵》《续集》（旧作合集） | 1983-1988年 | 晚年隐居洛杉矶，几乎与世隔绝。 | 整理出版旧作，张爱玲作品在台港持续出版，形成“张爱玲热”。 |
| 《对照记》（散文集） | 1994年 | 晚年的最后一部作品，整理家族相册。 | 由皇冠出版。 |
| 《小团圆》（长篇小说） | 1970年代创作，2009年出版 | 自传体小说，遗作。 | 1995年9月8日于洛杉矶西木区公寓逝世，享年74岁。骨灰遵遗嘱撒入太平洋。 |
| 《雷峰塔》《易经》（英文自传体小说） | 2010年出版 | 1960年代用英文写作的自传体小说。 | 遗作整理出版，为研究张爱玲提供新材料。 |
| 《少帅》（长篇小说） | 2014年出版 | 以张学良为原型的长篇小说，1960年代创作。 | 遗作整理出版，引发新一波张爱玲研究热潮。 |
